# horoco.co

Static website for **horoco LLC**, hosted on GitHub Pages at [www.horoco.co](https://www.horoco.co).

## Structure

- `index.html` — Main landing page
- `privacy.html` — Privacy Policy
- `support.html` — Support & FAQ
- `style.css` — Styles
- `CNAME` — GitHub Pages custom domain config
- `logo.png` — Company logo (add your logo file here)

## Deployment

This site is deployed automatically via GitHub Pages from the `main` branch. Ensure GitHub Pages is enabled in the repository settings and the custom domain `www.horoco.co` is configured with appropriate DNS records.

### DNS Setup

Point your domain to GitHub Pages:
- Add a CNAME record for `www` pointing to `<username>.github.io`
- Optionally add A records for the apex domain (`horoco.co`) pointing to GitHub's IPs
